# AI-Powered SEO Content Production Research

## Overview

This repository contains research on AI-powered SEO content production for B2B SaaS companies.

The objective is to identify leading practitioners, collect their latest insights from LinkedIn and YouTube, and organize the findings into a structured knowledge base.

## Repository Structure

```
research/
├── sources.md
├── linkedin-posts/
├── youtube-transcripts/
└── other/
```

## Goal

- Identify 10 industry experts
- Collect recent LinkedIn posts
- Collect YouTube transcripts
- Organize research into Markdown files
- Build a reusable knowledge base
